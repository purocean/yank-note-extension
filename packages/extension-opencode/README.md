# OpenCode

[OpenCode](https://opencode.ai/) is an AI coding agent built for the terminal.

This extension allows you to invoke OpenCode directly from within Yank Note, making it easy to use AI-powered assistance for your documents.

![img](https://github.com/user-attachments/assets/74ab43b5-4838-4199-ba41-5b6563d091c5)

## Features

- Quick access to OpenCode from the tab bar
- Expandable panel interface

## Requirements

- [OpenCode](https://opencode.ai/docs/) must be installed on your system
- API keys for your preferred LLM provider

## Usage

1. Click the OpenCode icon in the tab bar
2. The OpenCode panel will open
3. Use the expand/collapse button to resize the panel
