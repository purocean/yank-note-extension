# Review Comments

Review Comments lets readers select text in Yank Note's rendered preview, attach a comment, and keep both the target context and the comment readable in the Markdown document.

## Usage

1. Select text in the default preview.
2. Click the floating comment button or run the **Add review comment** action.
3. Enter a comment and save it with `Ctrl/Cmd + Enter`.
4. Use the icon controls in the document's review section to locate, edit, or delete a comment.

The extension stores review comments after a horizontal rule at the end of the document using readable containers, blockquotes, and stable style classes. The original article text is not modified.
