# Sidebar Terminal

Run shell commands in a dedicated Yank Note sidebar terminal.

![Sidebar Terminal](https://github.com/user-attachments/assets/08a2d224-e044-465c-a0c4-7a59416bac1d)

## Features

- Open a terminal in the Yank Note sidebar without leaving the editor.
- Start a regular shell session in the current repository.
- Launch common CLI tools from quick command buttons.
- Insert the current document path into the terminal input.
- Stop and restart terminal sessions from the panel controls.

## Usage

1. Click the `Sidebar Terminal` icon in Yank Note.
2. Click `Start Terminal` to open a shell session.
3. Use a quick command button to launch a configured CLI command.
4. After the terminal starts, use the panel controls to stop the session or insert the current document path.

## Requirements

- Yank Note `>= 3.87.0`
- Terminal support enabled in Yank Note
