# Sidebar Terminal

Sidebar Terminal is a Yank Note extension for running shell commands in a dedicated terminal panel.

## Features

- Quick access from the tab bar
- Expandable panel interface
- Proxy environment support for terminal sessions

## Requirements

- Yank Note with terminal support enabled

## Usage

1. Click the Sidebar Terminal icon in the tab bar.
2. Open the terminal panel.
3. Start a shell session in the current workspace.
